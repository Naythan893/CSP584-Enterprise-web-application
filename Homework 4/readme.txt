Smart Portables

1.Go to the TomCat Server downloaded file folder 
2.Navigate to the class file folder Store\WEB-INF\classes
3.Run the java files javac *.java
4.Start MySQL Server
Schema Name : egdb
Tables - orders, user, productdetails.
orders - Order and transaction details
user - username, password and user related details
product details - all product related details including product name,id, price, etc.
5. start Mongod.exe and mongo.exe
6. go to browser and type http://localhost/WebStore/
